⚠️ Note on Persistence

Content changes are stored in memory on the server and will reset when the serverless function instance is recycled (cold start). The resume page also caches the latest values in localStorage so your browser will always show your saved content.

For permanent cross-user persistence, connect Vercel KV and update api/_store.js to read/write from KV.
