Data Management

Analytics data is stored locally in this browser's localStorage. It is only visible on the device and browser where visits were recorded.

How Analytics Works

Each visit to your portfolio is tracked and the following data is stored:

  • IP address (via ipapi.co)
  • Country & city (geolocation)
  • Visit timestamp
  • Referrer URL or "Direct"
  • Browser & device type (from User-Agent)

Storage limit: The last 500 visits are kept. Data is only visible on the same browser/device.
